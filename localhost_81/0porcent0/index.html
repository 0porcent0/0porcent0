HOW DOES THIS WEBSITE WORK?

The aim of this platform is to create dialogues around the showed issues. You can find some of the previous inputs or see all of them clicking on SEE ALL. You’ll find more interacting spaces around the website. Feel free to interact with them. In order to keep this platform a safe space that respects the collective freedom of expression, this website is moderated by Ana Leite and Inês Mendes.
